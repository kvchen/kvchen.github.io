Question 1

Suppose we’ve run this code in Python:

class ClassA:
    a = 0
    c = 10

    def __init__(self):
        a = 1
        self.a = 2


class ClassB(ClassA):
    b = 1
    c = 20

    def __init__(self):
        ClassB.b += 1
        self.b = ClassB.b

What would the Python interpreter display for each of the following lines?

>>> ClassA.a

>>> a = ClassA()
>>> [a.a, ClassA.a, a.c]

>>> [ClassB.b, ClassB.a, ClassB.c]

>>> b = ClassB()
>>> [b.b, ClassB.b]

>>> b = ClassB()
>>> [b.b, ClassB.b]
