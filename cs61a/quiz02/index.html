Question 3

Suppose we’ve already entered the following code into the Python interpreter:

wild = 2

def pig(out):
    while out < 10:
        if out % 2:
            print('oink')
        if out % 3:
            print('squeal')

        def x(x):
            return out + x

        out = out * out
    return x(out)

def hog(wild):
    while pig(wild):
        wild = wild // 3

        if wild % 3 == 0:
            return pig(wild + 3)

def swine(hog):
    return hog(wild) + wild

What would Python print for each of the following lines?

>>> pig(3)

>>> y = hog(7)

>>> hog(swine(hog))

>>> swine(pig)
